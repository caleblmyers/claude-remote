# Claude Knowledge Base

This directory stores reference material for Claude Code across sessions.
It is NOT application code — it's a knowledge base for AI-assisted development.

## Structure

- `app-overview.md` — How Claude Remote works: architecture, data flow, key files
- `costs.md` — AI/infra cost tracking and estimates
- `errors.md` — Error log: recurring issues and their solutions
- `decisions.md` — Architectural decisions and their rationale
- `todos.md` — Deferred ideas, work sets, and future improvements
- `changelog.md` — Session-by-session development log

## Usage

These files are consulted during development to avoid repeating mistakes,
recall prior decisions, and build on accumulated knowledge.
